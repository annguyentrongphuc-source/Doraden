# Audio Placement Guide

## Music
Place background music files here:

```
bgm_school.ogg        — Upbeat school theme
bgm_mei_room.ogg      — Mei's room, melancholic/chill
bgm_tension.ogg       — Test/crisis scenes
bgm_emotional.ogg     — Emotional moments (Pi singing, Hương confession)
bgm_night.ogg         — Late night, clone account browsing
bgm_sunrise.ogg       — Dawn/ending theme
bgm_flashback.ogg     — Flashback scenes
bgm_credits.ogg       — Credits music
```

## Sound Effects
```
alarm.ogg              — Morning alarm clock
school_bell.ogg        — School bell
camera_click.ogg       — Camera shutter (for Vy's magic camera)
phone_buzz.ogg         — Phone notification
typing.ogg             — Keyboard typing
door_open.ogg          — Door opening
rain.ogg               — Rain ambient
wind.ogg               — Wind ambient
heartbeat.ogg          — Heartbeat (anxiety moments)
```

## Format: OGG Vorbis recommended (best Ren'Py compatibility)
