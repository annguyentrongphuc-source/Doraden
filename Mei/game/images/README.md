## Backgrounds
Place backgrounds in a `bg/` subfolder:

```
bg/school_exterior.png
bg/classroom.png

bg/school_hallway_evening.png
bg/school_yard.png

```

## Recommended resolution: 1920x1080

Ren'Py will auto-detect images matching character tag names.
For example, `show mei happy` will look for `mei happy.png`.
